Installation
============

Preparing for sidpy
-------------------
`sidpy <https://github.com/pycroscopy/sidpy>`_ depends on a fairly standard scientific Python stack.
We now recommend `uv <https://docs.astral.sh/uv/>`_ for creating and managing the project environment.
It can create the virtual environment, resolve dependencies from ``pyproject.toml``, and run commands inside the project env.

If you do not already have ``uv`` installed, follow the `official installation instructions <https://docs.astral.sh/uv/getting-started/installation/>`_.

Installing sidpy
-----------------
1. Open a terminal window.
2. From the repository root, create or update the environment:

   .. code:: bash

      uv sync --all-extras

3. Run the test suite or other commands inside the environment:

   .. code:: bash

      uv run pytest

If you want to create a standalone virtual environment before syncing dependencies, you can do that too:

.. code:: bash

   uv venv --python 3.10
   uv sync --all-extras

Installing from PyPI
--------------------
To install a published release into the active environment:

.. code:: bash

   uv pip install sidpy

Updating sidpy
--------------
If you are working from a git checkout, refresh dependencies and the lock state with:

.. code:: bash

   uv sync --all-extras

If you are installing from PyPI and want the latest release:

.. code:: bash

   uv pip install --upgrade sidpy

Other software
--------------
We recommend `HDF View <https://support.hdfgroup.org/products/java/hdfview/>`_ for exploring HDF5 files generated by and used in sidpy.
